# Static assets (served at the site root)

## Brand logo

Save the TRAP MADE IT logo here as:

```
public/trap-logo.png
```

- **White artwork on a black (or transparent) background** — the game keys the
  black out automatically, so only the white mark shows on the wall.
- A transparent PNG works too and is slightly cleaner.
- Landscape, ~1000px+ wide keeps it crisp on the walls.

Once the file is here, hard-refresh the game — it appears in 3D on the walls in
place of the old gold "TRAP" text. If the file is missing, the game falls back
to that text, so nothing breaks.
