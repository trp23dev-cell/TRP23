# Garment images

Drop the real high-res product photos here and the game will use them
automatically (falling back to the low-res images baked into the code if a file
is missing). **Refresh with a hard reload after adding files.**

## Best results
- **Front/back only** — one garment per image, roughly centred, shot straight-on.
- **Transparent PNG** (background already removed) is ideal — the game detects
  the transparency and uses it as-is, no auto-cut needed. Sharpest result.
- If the background is still there, a **plain dark or plain white** backdrop cuts
  cleanest. Busy/gradient backgrounds are harder to key out.
- Aim for ~800px tall or more so it stays crisp on the mannequins.
